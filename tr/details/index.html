PermalinkDetails


Unity API uses hard-coded values and they are generally not good. Because when you change or remove a hard-coded value in Unity Editor, you will not notify if the hard-coded value used in many different location of your project code. The beauty of Hard-Coded Generator is when you modify a hard-coded value, Hard-Coded Generator will notify you of any misspellings or missing references by the Unity console.
